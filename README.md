# Expense Health Checker

月ごとの支出をカテゴリごとに記録し、  
予算に対する消費率や家計の健全度をチェックできる **Pythonの簡易家計簿アプリ**です。

---

## 概要

このプログラムでは以下のことができます。

- カテゴリごとの支出入力  
- カテゴリの追加  
- 予算に対する消費率の計算  
- 各カテゴリの家計状態判定  
- 全体の家計状態判定  
- 支出が最も多いカテゴリの表示  
- 危険カテゴリのアドバイス表示  

---

## 使用技術

- Python  
- 辞書（dictionary）  
- 関数  
- whileループ  
- 入力バリデーション

## 出力例


食費：支出12000円・予算30000円・残金18000円・消費率40％・判定健全
日用品：支出3000円・予算10000円・残金7000円・消費率30％・判定健全
娯楽：支出10000円・予算15000円・残金5000円・消費率66％・判定注意
交通費：支出8000円・予算10000円・残金2000円・消費率80％・判定注意
その他：支出0円・予算20000円・残金20000円・消費率0％・判定健全


---

### 全体判定


総支出33000円・総予算85000円・全体判定健全


---

### 最大支出カテゴリ


一番支出の多いカテゴリは娯楽です（10000円）


---

### アドバイス


来月は娯楽、交通費カテゴリの支出を抑えましょう


または


危険カテゴリなし


---

## 判定基準

### カテゴリ判定

| 条件 | 判定 |
|-----|-----|
| 支出 > 予算 | 危険 |
| 消費率70%以上 | 注意 |
| それ以外 | 健全 |

※交通費は80%以上で注意

---

### 全体判定

| 消費率 | 判定 |
|------|------|
| 70%以下 | 健全 |
| 70%〜100% | 注意 |
| 100%以上 | 危険 |
